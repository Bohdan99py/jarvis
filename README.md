## Команды

Системные команды обрабатываются локально, без API, бесплатно:

`привет` · `время` · `дата` · `блокнот` · `калькулятор` · `проводник` · `браузер` · `запусти <app>` · `найди <запрос>` · `youtube <запрос>` · `напечатай <текст>` · `нажми <клавиша>` · `комбо ctrl+c` · `заблокируй` · `запомни ключ=значение` · `вспомни <ключ>` · `память` · `статистика` · `обновление` · `плагины` · `помощь`

Всё, что не попало в список — уходит в Claude API как свободный вопрос.

---

## Сценарии (в разработке)

Джарвис адаптируется под пользователя через систему сценариев:

**Разработчик игр** — интеграция с Unreal Engine 5, вайбкодинг через Claude, мониторинг логов, управление билдами.

**Геймер** — оверлей статистики, гайды не выходя из игры, запись клипов голосом, управление Discord.

**QA Тестировщик** — автоматические баг-репорты, анализ упавших тестов, утренний дайджест CI.

**Художник** — рисование в Photoshop/Krita голосом, генерация изображений, голосовые команды в Blender.

Один движок — разные личности. Каждый пользователь получает своего Джарвиса.
