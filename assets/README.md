# Assets

Файлы для установщика (Inno Setup):

| Файл | Размер | Описание |
|------|--------|----------|
| `icon.ico` | 256x256 | Иконка приложения (формат .ico, мульти-размер) |
| `wizard_image.bmp` | 164x314 | Картинка слева в установщике |
| `wizard_small.bmp` | 55x58 | Маленькая иконка в заголовке установщика |

## Как создать icon.ico

1. Нарисуй или сгенерируй иконку 256x256 PNG
2. Конвертируй в .ico: https://icoconvert.com/
3. Положи сюда как `icon.ico`

Без этих файлов установщик соберётся, но без кастомных картинок.
